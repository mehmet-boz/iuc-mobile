# IUC Mobile

Istanbul University-Cerrahpasa — Mobile Application Development course.

This repository contains weekly hands-on projects built with [Expo](https://expo.dev) and React Native.

## Structure

| Folder | Description |
|--------|-------------|
| `week-01/` | Intro to Expo & React Native — first app |

## Running a project

```bash
cd week-01
npm install
npm start
```

Then scan the QR code with the [Expo Go](https://expo.dev/go) app on your phone, or press `i` for iOS simulator / `a` for Android emulator.

## Requirements

- Node.js 18+
- npm 9+
- Expo Go app on your device (or a simulator)
